1. 네트워크 관련 신기술
- 1. IoT (사물인터넷) -> 실세계와 가상 세계, 인터넷으로 연결, 서비스 기반 기술
- 2. M2M (사물 통신) -> 무선 통신 기계 & 기계, 감시
- 3. 모바일 컴퓨팅 -> 휴대용 기기의 자유롭게 네트워크 접속, 업무 처리
- 4. 클라우드 컴퓨팅 -> 컴퓨팅 자원 중앙화, 인터넷을 통해 컴퓨터 작업 수행 가능한 가상화 환경
- 5. 그리드 컴퓨팅 -> 지리적 분산, 초고속 인터넷망으로 연결
- 6. 모바일 클라우드 컴퓨팅 -> ICT 기술, 소비자와 소비자의 파트너, 클라우드 서비스 이용
- 7. 인터클라우드 컴퓨팅 -> 클라우드 서비스 연동, 컴퓨팅 자원의 동적 할당 가능
- 8. 메시 네트워크 (Mesh Network) -> 차세대 이동통신, 대규모 디바이스, 공공안전, 홈네트워킹
- 9. Wi-Sun (와이선) -> 스마트 그리드, 저전력 장거리 (LPWA)
- 10. NDN -> 콘텐츠 자체 정보 & 라우터 기능만으로 데이터 전송 수행, 기존 IP 망을 대체할 아키텍처
- 11. NGN (차세대 통신망) -> ITU-T 개발, 이동 사용자, 완전한 이동성 (Full Mobility) 제공 목표
- 12. SDN (S/W 정의 네트워킹) -> 네트워트 컴퓨터처럼 모델링, 네트워킹 가상화하여 제어 및 관리 네트워크
- 13. NFC (정의 네트워크) -> 고주파 (HF), 아주 가까운 거리에서의 통신 (RFID)
- 14. UWB (초광대역) -> 무선 디지털 펄스, 0.5m/W
- 15. 피코넷 (PICONET) -> 블루투스 기술, UWB 기술 활용, 통신망 형성
- 16. WBAN -> 웨어러블 (Wearable), 몸에 심는 (Implant) 형태, 무선 개인 영역 네트워킹
- 17. GIS (지리 정보 시스템) -> 지리적인 자료 수집, 저장, 분석, 출력 (컴퓨터 응용 시스템), 위성 활용
- 18. USN (유비쿼터스 센서 네트워크) -> 각종 센서로 수집한 정보를 무선으로 수집, RFID 부착
- 19. SON (자동 구성 네트워크) -> 주변 상황, 스스로 망 구성, 통신망 커러비지 및 전송 용량 확장
- 20. 애드훅 네트워크 (Ad-hoc Network) -> 재난 현장 모바일 호스트, 단기간 사용
- 21. 네트워크 슬라이싱 -> 코어 네트워크 => 가상 네트워크, 3GPP, 5G(IMT-2020)의 핵심기술
- 22. 저전력 블루투스 기술 -> 2.4GHz 주파수 대역 사용, 연결 대기 상태에서는 절전모드
- 23. 지능형 초연결망 -> 스마트 시티
- 24. 파장분할 다중화 (WDM) -> 광섬유, 파장 다른 광선 간섭 금지
- 25. 소프트웨어 정의 데이터센터 (SDDC) -> 데이터 센터 모든 자원 가상화, 다양한 소프트웨어 정의 기술
- 26. 개방형 링크드 데이터 (LOD) -> Linked Data + Open Data, 개별 URI (인터넷 식별자)


2. SW 관련 신기술
- 1. 인공지능 -> 컴퓨터 스스로 추론, 학습, 판단
- 2. 뉴럴링크 -> 테슬라
- 3. 딥러닝 -> 인공 신경망 (ANN)
- 4. 전문가 시스템 -> 특정 분야의 전문가
- 5. 증강현실 (AR) -> 혼합 현실 (MR)
- 6. 블록체인 -> P2P 네트워크, 네트워크 참여자 (Peer)
- 7. 분산 원장 기술 (DLT) -> 중앙 관리자 & 데이터 저장소 X, P2P 망, 모든 거래 목록 분산 저장
- 8. 해시 (Hash) -> 입력 데이터, 메시지를 고정된 길이의 값이나 키로 변환
- 9. 양자 암호키 분배 (QKD) -> 비밀키 분배 관리
- 10. 프라이버시 강화 기술 (PET) -> 개인정보 침해 위험 방지
- 11. 공통 평가 기준 (CC) -> ISO 15408, 정보의 순기능
- 12. 개인정보 영향평가 제도 (PIA) -> 개인정보 활용, 국민의 사생활에 미치는 영향
- 13. 그레이웨어 (Grayware) -> 소프트웨어 제공자 악의적 x, 사용자 입장에서는 유용 OR 악의적, 애드웨어, 트랙웨어, 기타 악성 코드
- 14. 매시업 (Mashup) -> 웹 제공 서비스 및 정보, 새로운 소프트웨어나 서비스, DB 등을 만드는 기술
- 15. 리치 인터넷 애플리케이션 (RIA) -> 기존 HTML 보다 역동적, 인터랙티브한 웹 페이지 제공
- 16. 시맨틱 웹 -> 컴퓨터가 사람 대신 정보 읽고 이해
- 17. 증발품 -> 판매 계획 O, 실제 판매 X
- 18. 오픈 그리드 서비스 아키텍처 (OGSA) -> 애플리케이션 공유를 위한 웹 서비스, 그리드 상에서 제공
- 19. 서비스 지향 아키텍처 (SOA) -> 기업 정보 시스템 공유, 표현, 업무 프로세스, 서비스 중간, 애플리케이션, 데이터 저장
- 20. 서비스형 소프트웨어 (SaaS) -> 사용자가 필요로 하는 서비스만 이용 가능
- 21. 소프트웨어 에스크로 -> S/W 개발자 지식재산권 보호, 유지보수, 제3의 기관에 보관
- 22. 복잡 이벤트 처리 (CEP) -> 실시간 발생하는 많은 사건, 사건 발생 조건 정의
- 23. 디지털 트윈 (Digital Twin) -> 현실 속의 사물을 S/W 로 가상화한 Model


3. HW 관련 신기술
- 1. 고가용성 -> 안정적인 서비스 운영 (장시간), 장애 발생 시, 즉시 다른 시스템으로 대체 가능한 환경 구축 매커니즘
- 2. 3D Printing -> 평면 출력 X, 손으로 만질 수 있는 실제 물체로 만들어 내는 것
- 3. 4D Printing -> 특정 시간 or 조건 -> 자가 조립 (Self Assembly) 기술
- 4. RAID -> 데이터 블록, 디스크 속도 매우 향상
- 5. 4K 해상도 -> 차세대 고화질 모니터
- 6. 앤 스크린 (N-Screen) -> N개의 서로 다른 단말기에서 동일한 콘텐츠 이용 가능 서비스
- 7. 컴패니언 스크린 -> 방송 내용 공유, N-Screen 종류 중 하나, Second Screen
- 8. 신 클라이언트 PC (Thin Client PC) -> 하드디스크, 주변 장치 X, 기본적인 메모리만 있음, 서버 + 네트워크, 개인용 PC
- 9. 패블릿 (Phablet) -> Phone + Tablet, 5인치 이상의 대화면 스마트폰
- 10. C형 USB -> 범용 인터페이스 규격인 USB 표준 중 하나
- 11. 멤스 (MEMS) -> 초정밀 반도체 제조 기술, 센서, 액추에이터 (Actuator)
- 12. 트러스트존 기술 (TrustZone Technology) -> 하나의 프로세서, 일반 구역, 보안 구역
- 13. 엠디스크 (M-DISC) -> 한 번의 기록 만으로 자료 영구 보관
- 14. 멤리스터 (Memristor) -> Memory + Register, 기존 경험을 모두 기억하는 특별한 소자


4. DB 관련 신기술
- 1. 빅데이터 -> 막대한 양의 정형 또는 비정형 데이터 집합, 기업과 정부, 포털 등의 빅데이터 효과적 분석, 미래 예측
- 2. 브로드 데이터 -> 소비자와 상호작용을 통해 생성된 것, 기업 마케팅에 효율적, 새로운 데이터 & 기존 데이터 + 새로운 가치
- 3. 메타 데이터 -> 일련의 데이터를 정의하고 설명해 주는 데이터, 데이터 사전의 내용, 스키마
- 4. 디지털 아카이빙 -> 디지털 정보 자원의 장기적 보관, 아날로그 콘텐츠 => 디지털 변환, 압축 저장, 디지털 콘텐츠 체계쩍 분류하고 메타 데이터 생성, DB
- 5. 하둡 -> 오픈 소스 기반 분산 컴퓨팅 플랫폼, 구글 및 야후 적용
- 6. 맵리듀스 -> 대용량 데이터 분산 처리, 데이터 분류 Map 작업 & 원하는 데이터 추출 Reduce 작업,  Google 고안
- 7. 타조 -> 하둡 기반 분산 데이터 웨어하우스 프로젝트
- 8. 데이터 다이어트 -> 데이터 압축, 중복된 정보 배제, 새로운 기준에 따라 나누어 저장
- 9. 데이터 마이닝 -> 대량 데이터 분석, 상호 관계 규명, 일정 패턴 찾아내는 기법
- 10. OLAP -> 다차원으로 이루어진 데이터, 통계적인 요약정보 분석, Roll-up, Dril-Down, Dill-Through, Dril-across, Pivoting, Slicing, Dicing
