<스위치>
1. L2 -> OSI 2계층, MAC 주소 기반으로 프레임 전송, 동일 네트워크
2. L3 -> OSI 3계층, L2 + 라우터 기능, IP 주소 기반으로 패킷 전송, 서로 다른 네트워크
3. L4 -> OSI 4계층, L3 + 로드밸런서(특정 서버에만 부하 방지), IP 주소 & TCP/UDP 기반, 트래픽 분배
4. L7 -> OSI 7계층, L2 + L3 + L4 -> IP 주소 + TCP/UDP 포트 정보 + 패킷 내용, 세밀한 로드밸런싱


<백본 스위치>
L3 스위치, 네트워크들 연결 시 중추적 역할, 모든 패킷이 지나가는 네트워크 중심에 배치


<Hierachical 3 Layer 모델>
1. 엑세스 계층 -> 사용자 네트워크 최초 접속 시점, 사용자 통신, 포트 수 = 사용자 수, L2 스위치
2. 디스트리뷰션 계층 -> 엑세스 계층 장치들 연결, LAN 간 라우팅 기능 수행, 라우터 & L3 스위치
3. 코어 계층 -> 디스트리뷰션 계층 통신 인터넷에 연결 (백본 계층), 백본 스위치, 전자우편, 인터넷접속
